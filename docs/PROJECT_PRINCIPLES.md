# Project Principles: 昭和社員転生診断

最終更新日: 2026-05-06
管理者: ユーザー

## このツールは何のためのものか

令和のユーザーが昭和の会社に転生したら、どんな社員アバターになるかを判定するバラエティ系ネタ診断です。

正式タイトル:

- 転生したら昭和の社員だった件

診断名/共有用名称:

- 昭和社員転生診断

目指すこと:

- 飲み会やSNSで「これお前じゃん」と笑える診断にする。
- タイプ名は尖らせつつ、本文では強みや笑えるポイントに変換する。
- メイン16タイプに、サブ罪状、令和NGタグ、NG濃度でバリエーションを出す。

目指さないこと:

- 現代の能力、人格、職業適性を決めること。
- 現実の人事評価や採用判断に使える診断にすること。
- 現代の本人を直接攻撃すること。

## 優先する価値

1. SNSで共有したくなるネタ感
2. 「ちょっと刺さるけど笑える」バランス
3. 昭和転生アバターとしての安全な距離
4. 16タイプ構造を保った実装のシンプルさ

## 安全ライン

- 現代の本人を「パワハラ」「無能」「老害」などと直接断定しない。
- あくまで昭和に転生した架空アバターとして扱う。
- 結果文には必ず「令和では封印したいムーブ」「令和への帰還メモ」で救いを入れる。
- タイプ名は尖らせるが、本文では強みや昭和なら評価されたポイントに変換する。
- 現実の人事評価、適職診断、能力診断に見せない。

## 壊してはいけない前提

- メイン結果は16タイプ固定。
- 現行の4軸、12問、各軸3問構造を活かす。
- ユーザーにはタイプコードや軸名を前面に出しすぎない。
- `src/diagnosisData.ts` が診断データの正本。
- `src/diagnosisLogic.ts` がスコア計算と共有文の正本。
- `src/App.tsx` が画面本体。

## 今はやらないこと

- 16タイプ以上に増やすこと。
- 静的な16枚共有カード制作。
- 生成キャラクター画像の大量制作。
- 公開前Launch Readiness。

今は、16タイプと大量サブ罪状でバラエティ感を出すことを優先します。

# 転生したら昭和の社員だった件

令和のユーザーが昭和の会社に転生したら、どんな社員アバターになるかを判定するバラエティ系ネタ診断です。現行の4軸・16タイプ・12問構造を活かしつつ、結果は「昭和社員転生診断」として表示します。

## 方針

- 現代の能力、人格、職業適性を決める診断ではありません。
- ありえない昭和職場を舞台にしたネタ診断です。
- タイプ名は尖らせつつ、本文では強みや令和への帰還メモで救いを入れます。
- メイン結果は16タイプ固定、サブ罪状、令和NGタグ、NG濃度でバリエーションを出します。

## 主要ファイル

- `src/App.tsx`: イントロ、設問、転生結果、共有カード保存、X共有を構成する画面実装。
- `src/diagnosisData.ts`: 4軸、12問、16タイプ、サブ罪状、軸別アドバイス。
- `src/diagnosisLogic.ts`: スコア計算、診断コード生成、関連タイプ、共有テキスト。
- `src/styles.css`: UIテーマとレスポンシブスタイル。
- `assets/work-style-hero.png`: ファーストビュー用の生成ビジュアル。
